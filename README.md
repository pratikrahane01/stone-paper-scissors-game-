## 🚀 How to Play

1. Run the Python script in your terminal:
   ```bash
   python "stone,paper,scissor game.py"
   ```
2. Enter your choice: `stone`, `paper`, or `scissor`.
3. The computer will randomly make its choice.
4. See who wins!
